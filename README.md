# Dania Cup

Installable World Cup knockout bracket app with live updates and alerts.

Upload these files to the root of the GitHub repository, not inside a folder:
- index.html
- manifest.json
- service-worker.js
- README.md
- icons/

After uploading, refresh the site. If the old app icon still appears, uninstall/remove the old app shortcut from your phone and install it again from Chrome.
