# Dania Cup

Installable World Cup knockout bracket PWA.

## Update instructions
Upload the contents of this folder to the root of the GitHub repository:
- index.html
- manifest.json
- service-worker.js
- README.md
- icons folder

GitHub Pages URL:
https://ibrahimali0114.github.io/dania-world-cup/

Open the URL in Chrome and choose **Add to Home screen** / **Install app**.

Alerts for goals and full-time results work while the app is open after tapping **Enable alerts**.
