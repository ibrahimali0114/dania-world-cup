# Dania's World Cup Tracker

Live World Cup knockout bracket PWA.

## GitHub Pages setup
1. Upload these files to the root of the repository.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main, folder: /root.
5. Save.

Live URL should be:
https://ibrahimali0114.github.io/dania-world-cup/

To install as an app, open the live URL in Chrome and choose Add to Home screen / Install app.
