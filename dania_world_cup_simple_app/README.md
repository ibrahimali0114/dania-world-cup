# Dania's World Cup Tracker

Simple installable World Cup bracket app.

## Files
Upload these files/folders to the root of the GitHub repository:
- index.html
- manifest.json
- service-worker.js
- README.md
- icons/

Live URL:
https://ibrahimali0114.github.io/dania-world-cup/

Open the live URL in Chrome and choose Install app / Add to Home screen.
